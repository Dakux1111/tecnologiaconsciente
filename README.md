## 💡 About the Project
Tecnología Consciente is a web platform designed to share knowledge, reflections, and educational content about personal growth, awareness, and the impact of technology on human life.

## ⚙️ Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages (deployment)

## 🚀 Purpose
The goal of this project is to create accessible content that inspires people to reflect, learn, and use technology in a more conscious and meaningful way.
